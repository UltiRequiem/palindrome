# palindrome

> A word, phrase, or sequence that reads the same backward as forward.

## Usage

```typescript
import randomItem, {
  randomMultipleItems,
} from "https://deno.land/x/random_item/mod.ts";

randomItem(["ğŸ´", "ğŸ¦„", "ğŸŒˆ"]);
//=> 'ğŸ¦„'

randomMultipleItems(["ğŸ´", "ğŸ¦„", "ğŸŒˆ"], 2);
//=> ['ğŸŒˆ', 'ğŸ¦„']
```
